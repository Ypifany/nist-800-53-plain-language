# nist-800-53-plain-language
NIST SP 800-53 security controls in plain language, built by Cheryl Abram, Ypifany LLC
This is a single-file, interactive HTML tool covering all 20 control families from NIST SP 800-53 Rev 5.2. Open nist-plain-language.html in any browser, no server or install required.
Scope
The tool includes controls and control enhancements that are actually assigned to the NIST SP 800-53B Moderate impact security control baseline, the baseline most commonly referenced in federal ATOs, DoD packages, and FedRAMP Moderate authorizations. It does not include every control in the full 800-53 catalog, only the ones NIST itself has selected into that baseline. Control and enhancement selections were pulled directly from NIST's published OSCAL baseline profile for SP 800-53 Revision 5.2.0.

Two families work a little differently. Program Management (PM) and PII Processing and Transparency (PT) are not assigned to the low, moderate, or high security baselines at all, NIST addresses them separately at the organization level. For those two families, this tool uses the controls NIST assigns to the Privacy Baseline instead, since that is the baseline NIST applies to them.

Because this is a baseline, not the full catalog, an actual system may need additional controls layered on top depending on its own risk assessment, tailoring decisions, or agency-specific overlays. This tool is a starting reference, not a substitute for a system-specific control selection.
How to use it
Each control includes six tabs: Plain Language, What It Does, Everyday Analog, Where This Lives, Look For, and Console / STIG. Enhancements are listed under their base control and expand on click, with the same six tabs scoped to that enhancement.

Use the search bar to jump straight to a control number, such as AC-2, AU-6, or AC-6(1), or browse the sidebar by family.
